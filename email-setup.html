Troubleshooting

Common Issues:

  • EmailJS not working: Check your service ID, template ID, and public key
  • Gmail blocking: Enable "Less secure app access" in Gmail settings
  • CORS errors: Make sure you're testing on a proper web server, not file://
  • Form not submitting: Check browser console for JavaScript errors

Security Considerations

Important Security Notes:

  • Never expose your private keys in client-side code
  • Use EmailJS public key only (it's safe for client-side use)
  • Consider adding reCAPTCHA to prevent spam
  • Set up rate limiting on your EmailJS account
